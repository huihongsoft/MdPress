Tab 步骤

使用 kind: tab 展示多平台、多语言等可切换内容。

List 步骤

使用 kind: list 展示线性执行的任务列表。

1

安装依赖

在项目根目录运行以下命令安装依赖： ```bash npm install ```

如果你更喜欢 pnpm，可以使用： ```bash pnpm install ```

pnpm 官方文档 →

或者选择 Yarn： ```bash yarn ```

! 请选择你常用的包管理器（Node.js >= 16）
2

启动与构建

  1. 启动开发环境
    运行开发服务，并在浏览器中预览： ```bash npm run dev ``` 打开终端中提示的本地地址（通常是 http://localhost:5173）。
  2. 运行 TypeScript 检查
    确保类型检查通过： ```bash npm run typecheck ```
  3. 构建生产包
    进行生产构建，验证构建是否成功： ```bash npm run build ```

上面的 Frontmatter 会在预览中渲染出：

  1. 顶部 Hero 区（如果你已经绑定了 hero 渲染逻辑）
  2. 一组特性卡片（来自 features）
  3. 一个 “安装依赖” 的 Tab Step，支持 NPM / pnpm / Yarn 三个切换选项
  4. 一个 “启动与构建” 的 List Step，用于指导用户完整走完本地开发到生产构建的流程
